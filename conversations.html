Conversations with Dashbud

Dashbud AI enables users to interact with their datasets through a conversational interface. Users can ask questions, receive AI-generated insights, and analyze structured query results.

1. Selecting a Dataset

Before interacting with Dashbud, users must select a dataset from the available imports. This dataset serves as the source for AI-driven analysis.

2. Chat Interface

Once a dataset is selected, users have two ways to interact with Dashbud:

  • Custom Queries: Users can type questions related to their dataset in the chatbox.
  • AI-Suggested Insights: Dashbud provides predefined analytical insights based on dataset trends.

3. AI-Suggested Insights

Dashbud automatically suggests key insights based on data patterns. Users can:

  • Analyze trends across different data points.
  • Explore regional, demographic, or temporal patterns.
  • Identify potential correlations within the dataset.

4. Viewing Structured Data Analysis

After selecting an AI-generated insight, Dashbud executes a query and presents the structured results in a readable format. Users can:

  • View numerical data breakdowns.
  • Compare results across different dataset attributes.
  • Use structured results to validate business insights.

5. Accessing SQL Queries

Each AI-generated insight is powered by an SQL query. Users can:

  • Copy Query: Extract the SQL statement for further manual analysis.
  • Modify Query: Adjust parameters to refine data exploration.
  • Export Results: Save structured insights for external reporting.
