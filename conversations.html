Step-by-Step Guide: Conversations with Dashbud

This section walks through an example of interacting with Dashbud AI to analyze datasets and generate insights.

Step 1: Selecting a Dataset

Users begin by selecting a dataset from the list of available imports. This dataset will be used as the basis for the AI-driven analysis.

Step 2: Opening the Chat Interface

Once a dataset is selected, users can either type custom queries into the chatbox or choose from AI-generated suggestions to explore trends.

Step 3: Receiving AI-Suggested Insights

Dashbud AI automatically suggests insights based on the dataset. These insights highlight key takeaways and patterns within the data.

Step 4: Viewing Structured Query Results

Users can view structured query results, which provide a breakdown of key metrics and trends in the dataset for deeper analysis.

Step 5: Accessing the SQL Query

For advanced users, Dashbud provides the SQL query behind the dataset insights. Users can view, copy, and modify the query to further refine their analysis.
