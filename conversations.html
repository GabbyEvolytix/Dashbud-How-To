Conversations with Dashbud

Dashbud is an AI-powered assistant within Evolytix that helps users interact with their datasets in a conversational way. This feature allows users to query datasets, gain AI-generated insights, and retrieve structured data analysis effortlessly.

1. Selecting a Dataset

Users start by choosing a dataset from their imported files. The selected dataset becomes the primary data source for the AI-powered chat, where queries can be made.

2. Engaging with the AI Chat

Once a dataset is selected, users can interact with Dashbud via chat. There are two ways to ask questions:

  • Manual Queries: Users type custom questions about the dataset.
  • AI-Suggested Insights: Dashbud provides pre-generated suggestions based on data trends.

3. AI-Generated Insights

Dashbud offers AI-driven insights by analyzing the dataset. These insights can highlight patterns, trends, and relationships between different data points.

4. Structured Data Analysis

Dashbud processes user queries and provides structured output. The results are displayed in a tabular format, offering a clear view of the datasetâ€™s key metrics.

5. SQL Query for Deeper Analysis

For advanced users, Dashbud generates SQL queries corresponding to the AI-generated insights. Users can view or copy these queries for further analysis or execution in a SQL environment.
