Import Data

This section guides you through importing data into Evolytix.

Click the "New Import" button to start the process.

Users will see the file selection interface where they can upload a CSV file.

After clicking on the upload area, the system file browser will appear to select a CSV file.

Users can preview the dataset before finalizing the import.

Once the file is successfully imported, users can view the structured schema or interact with Dashbud AI.
