Navigating the Datasets Page

This section walks through the different features of the datasets page.

Step 1: Opening the Datasets Page

When the user first clicks the "Datasets" option, this is the page they will see. It displays all available datasets with metadata such as the author and date modified.

Step 2: Accessing Dataset Options

Clicking on the three lines next to a dataset opens a menu with different options, including renaming, deleting, exporting, and viewing more details.

Step 3: Sharing a Dataset

Users can share their dataset to collaborate with others. This is the interface they see when selecting the "Share" option.

Step 4: Exporting a Dataset

To export a dataset, users can select the export option from the menu.

Step 5: Renaming a Dataset

Users can rename a dataset to improve organization. Clicking the rename option will prompt them to enter a new dataset name.

Step 6: Viewing Dataset Information

Clicking "More Info" allows users to view all the data inside a dataset, including the table structure and records.

Additional dataset information includes the query used to generate it and details about the dataset author.

Step 7: Deleting a Dataset

If a user chooses to delete a dataset, they will see a confirmation screen before finalizing, preventing accidental deletions.
